# KB Transport LLC Website

Professional dispatch and logistics marketing website.

## What’s included
- Single-page website (`index.html`) with responsive layout
- Sections: Hero, About, Services, Equipment, Why Choose Us, Process, Stats, Testimonials, Equipment Gallery, FAQ, Contact, Footer

## Deploy
This site is ready for **GitHub Pages**.

1. Create a repo (or use the existing one)
2. Push these files
3. Go to **Repo Settings → Pages**
4. Set the source to **GitHub Actions** or **Deploy from a branch** (commonly `main` / `/root`)
5. Wait for the deployment

## Local preview
Open `index.html` in a browser.

> Note: The contact form currently simulates submission on the client (no backend).
